# 株式会社衝動 公式ウェブサイト

株式会社衝動の公式ウェブサイトのソースコードです。

https://shodohq.com

## 技術スタック

- Next.js 15.4
- React 19
- TypeScript
- Tailwind CSS v4

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

http://localhost:3000 でアクセスできます。

## ビルド

```bash
npm run build
```

## デプロイ

GitHub Pagesへのデプロイは、mainブランチへのpush時に自動的に行われます。
